Principais formatações

Negritos / Destaque

Nesta frases temos um termo em negrito usando a tag B (não semantica. não tem significado, tem apenas forma)

Nesta frase, tenho um termo em destaque usando a tag STRONG (semantica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I ( não semantica).

Nesta frase temos um termo em ênsafe usando a tag EM (semantica)

Texto marcado

Podemos criar tambem um texto marcado usando a tag MARK

Podemos criar um STYLE Background-color dentro da tag mark e mudar a cor do marca texto como apresentado acima.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Usamos a tag BIG para texto grande e a tag SMALL para texto pequeno.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado usando a tag DEL, riscando assim o texto em questão como no exemplo.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag INS, sublinhando o texto em questão como apresentado no exemplo.

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag SUP, deixando o item marcado supeior como no exemplo.

Texto subscrito

Para inserir coisa do tipo H2O usando a tag SUB deixando o item marcado inferior como no exemplo.
