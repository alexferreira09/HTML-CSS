Listas ordenadas

Criamos listas ordenadas usando a tag OL e dentro da tag OL abrimos a tag LI para criarmos a lista ordenada que podem ter início com números, letras com algarismos romanos usando TYPE.

  1. Acordar
  2. Tomar café
  3. Ligar para João
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para a casa
  9. Jantar
  10. Dormir

Listas não ordenadas

Criamos listas não ordenadas usando a tag UL. Podemos alterar as bolinhas anteriores aos nomes usando a TYPE ( disc circle square ).

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão
