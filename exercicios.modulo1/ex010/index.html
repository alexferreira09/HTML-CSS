Usando links externos

Você pode acessar meu repositório público no GitHub

Você também pode acessar o meu canal do youtube

Voce poderia acessar meu instagram

Usamos a tag A para adicionarmos a URL do site para onde queremos ir.
Usamos TARGET="_BLANK" e também REL="EXTERNAL" para que se abra uma nova aba para o site em questao.

Recomenda-se abrir uma aba nova, quando você quer mandar para um site que não seja seu. Caso o link seja para um site que se esteja navegando dentro do seu próprio site, nao tem necessidade de abir uma nova aba.
